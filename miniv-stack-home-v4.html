Comandos de Operación

click para copiar
Levantar stack Docker completo
cd ~/Hermes-Stuff/deploy/codacus-local-ai-stack/ && docker compose up -d
Iniciar miniv-router (LLM)
systemctl --user start miniv-router.service
Iniciar results-server
systemctl --user start miniv-results-server.service
Iniciar Hermes API worker
systemctl --user start vector-miniv-api-worker.service
Iniciar Wiki-LLM
systemctl --user start wiki-llm-miniv.service
llama-server MANUAL (backup sin router)
# Matar router si está activo systemctl --user stop miniv-router.service # Lanzar llama-server directo (architect-35b-q6, flags del preset) ~/Hermes-Stuff/bin/llama-server-miniv-tq-vulkan \ --host 127.0.0.1 --port 9161 \ -m ~/models/gguf/35b-moe/Qwopus3.6-35B-A3B-v1-MTP-Q6_K.gguf \ --alias architect-35b-q6 \ -ngl 99 -c 131072 -b 2048 -ub 1024 -t 7 \ --cache-type-k turbo4 --cache-type-v turbo3 \ --flash-attn on --no-mmap --jinja \ --reasoning on --reasoning-budget 4096 \ --spec-type draft-mtp --spec-draft-n-max 2 --spec-draft-n-min 1 \ --metrics --slots --props
llama-server MANUAL (modelo simple sin MTP)
# Para modelos sin MTP (ornith, gemma4, etc.) ~/Hermes-Stuff/bin/llama-server-miniv-tq-vulkan \ --host 127.0.0.1 --port 9161 \ -m ~/models/gguf/ornith-1.0-35b/ornith-1.0-35b-Q6_K.gguf \ --alias ornith-35b-q6 \ -ngl 99 -c 65536 -b 2048 -ub 1024 -t 7 \ --cache-type-k turbo4 --cache-type-v turbo3 \ --flash-attn on --jinja --metrics --slots
Restaurar router (después de manual)
# Kill del proceso manual pkill -f "llama-server.*port=9161" sleep 5 # Reiniciar router systemctl --user start miniv-router.service
Chat completion (test router)
curl -s http://127.0.0.1:9161/v1/chat/completions \ -H "Content-Type: application/json" \ -d '{"model":"architect-35b-q6","messages":[{"role":"user","content":"Hola"}]}'
Cambiar modelo: cargar
curl -s -X POST http://127.0.0.1:9161/models/load \ -H "Content-Type: application/json" \ -d '{"model":"coding-coder-27b-q6"}'
Cambiar modelo: descargar
curl -s -X POST http://127.0.0.1:9161/models/unload \ -H "Content-Type: application/json" \ -d '{"model":"architect-35b-q6"}'
Listar modelos del router
curl -s http://127.0.0.1:9161/v1/models | jq .
